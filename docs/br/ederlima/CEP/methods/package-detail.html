A Classe AuthMethod é responsável por selecionar entre os métodos SIGNED ou UNSIGNED, permitindo ao usuário escolher entre usar ou não uma API Key fornecida pelo serviço, por padrão o método de autenticação selecionado é o AuthMethod.UNSIGNED.

A Classe ResponseMethod é responsável por selecionar entre os métodos XML_METHOD ou TEXT_METHOD permitindo ao usuário selecionar o método de conexão que mais se encaixa ao seu uso, por padrão o método selecionado é o XML_METHOD.
